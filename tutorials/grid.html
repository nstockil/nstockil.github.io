CSS Grid

Grid Container

  • the element with display:grid
  • can have many per page
  • can nest Containers

Grid Item

  • direct first level descendent of Grid Container
  • automatically placed in grid, or precisely defined in css
  • descendents of these items are not affected by the grid at all

Grid Line

  • Columns and rows of the grid
  • Have 4 lines out of the box. The four sides of the grid box
  • Additional lines are added through rows and column lines. This divides the grid into cells
  • Ordered by number starting with outer edges being number 1
  • This honours the alignement of the text/elements of the document
  • Can refer to a line by its number, or a defined name

Grid Cell

  • Space between two adjacent row and column lines (the intersection)
  • a.k.a. a table cell

Grid Track

  • Space between two or more adjacent gird lines
  • Column track - all vertical cells between two column lines
  • Row track - all horizontal cells between two row lines

Grid Area

  • Rectangular area between four specific grid lines
  • can cover multiple cells

Grid Gap

  • Empty space between grid tracks
  • Like a gutter between the tracks
  • As it is empty space, the background behind will show through
